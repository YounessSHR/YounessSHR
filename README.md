> Application web full-stack complète développée en 3 sprints agiles

**Stack :** `Spring Boot` `Java 21` `React` `MySQL` `JWT` `Spring Security` `iText 7` `Thymeleaf`

- API RESTful multi-rôles (Étudiant, Entreprise, Tuteur, Administration) avec authentification JWT
- Workflow complet : offres → candidatures → conventions (signature 3 parties) → suivi de stage
- Génération automatique de conventions PDF, notifications email asynchrones (`@Async`)
- Système de notifications in-app en temps réel, filtres avancés et pagination
